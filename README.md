# 世界遗产打卡 📍

一个帮助你记录和分享世界遗产探索之旅的 Web 应用。

## ✨ 特性

- 🌍 完整的世界遗产列表
- ✅ 打卡记录功能
- 📊 进度统计
- 📱 **PWA 支持 - 可离线使用**
- 🎨 简洁美观的界面
- 💾 数据本地存储

## 🚀 快速开始

### 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问 http://localhost:5173
```

### 构建

```bash
# 构建生产版本（包含 PWA 功能）
npm run build

# 预览生产版本
npm run preview
```

## 📦 PWA 功能

这个应用支持 PWA（渐进式 Web 应用），提供类似原生应用的体验：

- **离线访问**：首次访问后，即使没有网络也能使用
- **安装到桌面**：可以像原生应用一样安装到手机主屏幕
- **自动更新**：应用会在后台自动检测和更新到最新版本
- **Vercel 访问不稳定？**：PWA 的离线功能完美解决这个问题

详细说明请查看 [PWA_GUIDE.md](./PWA_GUIDE.md)

## 🔄 微信小程序迁移

当前的 PWA 实现方式对未来迁移到微信小程序非常友好：

- PWA 功能完全通过配置实现，不影响业务代码
- 数据存储使用标准 Web API，易于适配小程序
- 迁移时只需移除 PWA 插件配置即可

建议的迁移路径：
1. **使用 uni-app 重构** - 一套代码同时支持 H5 PWA 和微信小程序
2. **封装平台 API** - 为存储、网络等创建适配层
3. **渐进式迁移** - 先保持 PWA 版本，同步开发小程序版本

详细说明请查看 [PWA_GUIDE.md](./PWA_GUIDE.md#微信小程序兼容性考虑)
